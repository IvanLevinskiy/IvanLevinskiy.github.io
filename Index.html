ПРИМЕР ВЫПОЛНЕНЫХ РАБОТ:

HMI для рентгенологического аппарата
Система АСУТП осуществляет управление рентгенологическим аппаратом в металлургической промышленности. В качестве ПЛК используется Simatic S7-1500. Обмен данными между ПЛК и HMI осуществляется по Modbus TCP.
ПО для сбора данных
Программное обеспечение осуществляет опрос устройств по протоколам Simatic TCP и Modbus TCP. Значения процесса архивируются в базу данных. Опция: предоставление доступа к значениям процесса по технологии OPC UA.
Автоматизация линии сортировки
Система АСУТП осуществляет управление промышленной установкой - линией сортировки. В качестве ПЛК - S7-1200, программное обеспечение HMI написано на .NET (WPF). Взаимодействие между ПЛК и HMI осуществляется по протоколу Simatic TCP.
